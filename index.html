I have yet to move my hosting for this website to S3 and CloudFront, but in the meantime go to https://www.ryanmedhaug.io/ to check out my new weather station project.

For the project, I am using a LILYGO ESP32 microcontroller and DHT11 sensor to measure temperature and humidity data, send the data to Amazon's RDS, then using AWS's API gateway and Lambda to pull the data into a webpage. For the webpage, I am using the pulled sensor data in combination with Google Map's API. I plan on adding more functionality to the website as my free time permits :)
