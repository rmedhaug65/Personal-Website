Update

I have yet to move my hosting for this website from Github to S3 and CloudFront. In the meantime please go to https://www.ryanmedhaug.io/ to check out my new weather station project.

The purpose of the project was to get more familiar with AWS, IoT devices, and what building a pipeline looks like for developers. I plan on building out the website further and possibly look at adding an iOS/Android app. I would like to get more familiar with orchestrating containers and the monitoring and security surrounding them.

For the project, I am using a LILYGO ESP32 microcontroller and a DHT11 sensor to measure temperature and humidity data. I am sending the data to Amazon's SQL database, RDS, then using API gateway, in combination with Lambda, to pull the data from RDS into the webpage. For the webpage, the sensor data that is getting pulled combines with Google Map's API to plot the coordinates and display the data. Unfortunately, it is still a bit clunky as I work on my JS and React skills. I do plan on adding more functionality as my free time permits :)
